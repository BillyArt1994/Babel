## 高风险系统

### 瓶颈系统（多个系统依赖）
- **游戏循环管理器** - 9 个系统依赖，必须优先设计
- **技能系统** - 5 个系统依赖，复杂度高
- **塔建造系统** - 3 个系统依赖，核心机制

### 技术风险
- **对象池系统** - 性能关键，需要处理大量单位
- **敌人生成系统** - 难度曲线调优复杂

### 设计风险
- **升级系统** - 平衡性调试工作量大
- **技能系统** - 技能组合可能出现最优解

## 系统详情

### 1. 游戏循环管理器
**描述：** 管理游戏状态、15 分钟计时、胜利/失败条件、暂停/继续
**优先级：** MVP
**依赖：** 无
**被依赖：** 塔建造、敌人生成、技能、升级、所有 UI
**风险：** 高（瓶颈系统）

### 2. 输入系统
**描述：** 统一管理鼠标点击、键盘按键输入
**优先级：** MVP
**依赖：** 无
**被依赖：** 点击攻击、技能系统
**风险：** 低

### 3. 技能数据库
**描述：** CSV 配置表 + SkillParser 解析 + SkillDatabase 运行时字典（Trigger + Effect 参数）
**优先级：** MVP
**依赖：** 无
**被依赖：** 技能系统
**风险：** 低

### 4. 敌人数据库
**描述：** ScriptableObject 存储所有敌人类型属性（血量、速度、能力）
**优先级：** MVP
**依赖：** 无
**被依赖：** 敌人生成系统
**风险：** 低

### 5. 对象池系统
**描述：** 管理人类单位的创建/回收，优化性能
**优先级：** MVP
**依赖：** 无
**被依赖：** 敌人生成系统
**风险：** 中（性能关键）


### 6. 塔建造系统
**描述：** 倒金字塔 10 层结构、完整度进度、视觉呈现、失败条件
**优先级：** MVP
**依赖：** 游戏循环管理器
**被依赖：** 敌人 AI、摄像机、游戏 HUD
**风险：** 高（核心机制）

### 7. 敌人生成系统
**描述：** 敌人波次、生成频率、难度递增、单位类型选择
**优先级：** MVP
**依赖：** 游戏循环管理器、敌人数据库、对象池系统
**被依赖：** 敌人 AI 系统
**风险：** 中（难度曲线调优）

### 8. 点击攻击系统
**描述：** 纯执行层，被 DamageEffect 调用，4种攻击类型物理检测
**优先级：** MVP
**依赖：** 输入系统
**被依赖：** 粒子特效、音效系统
**风险：** 低

### 9. 技能系统
**描述：** Skill = Trigger + Effect 组合架构，SkillFactory 组装，EffectManager 持续效果管理
**优先级：** MVP
**依赖：** 游戏循环管理器、输入系统、技能数据库
**被依赖：** 升级系统、粒子特效、游戏 HUD、音效系统
**风险：** 高（复杂度高、平衡性）

### 10. 敌人 AI 系统
**描述：** 寻路、建造行为、特殊能力（治疗、加速等）
**优先级：** 核心版
**依赖：** 塔建造系统、敌人生成系统
**被依赖：** 无
**风险：** 中


### 11. 升级系统
**描述：** 信仰值获取、升级触发、3 选 1 技能选择
**优先级：** 核心版
**依赖：** 游戏循环管理器、技能系统
**被依赖：** 升级选择 UI
**风险：** 中（平衡性）

### 12. 粒子特效系统
**描述：** 神力技能视觉效果（雷击、洪水、地震等）
**优先级：** 核心版
**依赖：** 点击攻击系统、技能系统
**被依赖：** 无
**风险：** 低

### 13. 摄像机系统
**描述：** 视角跟随塔高度动态上升
**优先级：** 核心版
**依赖：** 塔建造系统
**被依赖：** 无
**风险：** 低

### 14. 游戏 HUD
**描述：** 显示时间、击杀数、塔进度、当前技能
**优先级：** 核心版
**依赖：** 游戏循环管理器、塔建造系统、技能系统
**被依赖：** 无
**风险：** 低

### 15. 升级选择 UI
**描述：** 3 选 1 技能选择界面
**优先级：** 核心版
**依赖：** 升级系统
**被依赖：** 无
**风险：** 低


### 16. 结算 UI
**描述：** 游戏结束统计（击杀数、塔高度、最接近成功的时刻）
**优先级：** 核心版
**依赖：** 游戏循环管理器
**被依赖：** 解锁系统、排行榜系统
**风险：** 低

### 17. 主菜单 UI
**描述：** 开始游戏、难度选择、退出
**优先级：** 完整版
**依赖：** 游戏循环管理器
**被依赖：** 无
**风险：** 低

### 18. 解锁系统
**描述：** 成就追踪、新神力解锁
**优先级：** 完整版
**依赖：** 游戏循环管理器、结算 UI
**被依赖：** 无
**风险：** 低

### 19. 排行榜系统
**描述：** 本地记录保存、最高分显示
**优先级：** 完整版
**依赖：** 游戏循环管理器、结算 UI
**被依赖：** 无
**风险：** 低

### 20. 音效系统
**描述：** 技能音效、环境音效、UI 音效
**优先级：** 完整版
**依赖：** 点击攻击系统、技能系统
**被依赖：** 无
