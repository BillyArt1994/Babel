# Unity Project Directory Structure

```
Babel/
├── Assets/
│   ├── Scripts/
│   │   ├── Core/           # 核心系统
│   │   ├── Gameplay/       # 游戏玩法
│   │   ├── UI/             # 用户界面
│   │   └── Utils/          # 工具类
│   ├── Prefabs/            # 预制体
│   ├── Scenes/             # 场景文件
│   ├── Materials/          # 材质
│   ├── Textures/           # 纹理
│   ├── Audio/              # 音频资源
│   └── Resources/          # 运行时加载资源
├── ProjectSettings/        # Unity 项目设置
├── Packages/               # Unity 包管理
└── .claude/
    ├── docs/               # 项目文档
    └── agents/             # 代理配置（可选）
```
