# 游戏项目初始化

引导用户完成以下步骤：

1. **游戏概念**
   - 游戏类型（动作、解谜、RPG 等）
   - 核心玩法机制
   - 目标平台

2. **技术栈确认**
   - Unity 版本
   - 渲染管线（URP/HDRP/Built-in）
   - 目标设备

3. **项目结构**
   - 创建核心文件夹
   - 设置场景模板
   - 初始化基础系统

4. **开发计划**
   - 里程碑规划
   - 优先级排序

开始询问用户游戏概念。
