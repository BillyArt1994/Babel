# 里程碑 1：MVP（最小可玩版本）

**目标：** 实现完整的核心游戏循环，可以从头玩到尾（胜利或失败）
**计划完成：** 2026-04-04
**冲刺：** Sprint 1

---

## 里程碑目标

玩家能够：
1. 启动游戏，看到人类从屏幕两侧涌入
2. 点击鼠标攻击人类
3. 观察巴别塔完整度随人类到达而上升
4. 触发升级选择（3 选 1）
5. 游戏在 15 分钟或塔完工时结束

|---|------|------|
| 1 | 游戏循环管理器 | 待开发 |
| 2 | 输入系统 | 待开发 |

| 6 | 塔建造系统 | 待开发 |
| 7 | 敌人生成系统 | 待开发 |
| 8 | 点击攻击系统 | 待开发 |
| 9 | 技能系统 | 待开发 |

## 不包含（推迟至里程碑 2）

- 敌人 AI 系统（无寻路，敌人直线前进）
- 升级系统（技能选择逻辑）
- 所有 UI 系统
- 视觉特效

## 成功标准

- [ ] 游戏可在 Unity 编辑器中无报错运行
- [ ] 人类可生成并移动至塔
- [ ] 点击可造成伤害并击杀人类
- [ ] 塔完整度正确累积
- [ ] 胜利/失败条件正确触发
- [ ] 信仰值累积并触发升级流程（MVP 版：升级界面可为占位符）
- [ ] 帧率在 100 个单位时保持 60fps+
